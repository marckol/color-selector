Sub palettes lists and grids cases

Lists

Two possibilities for list color selector:

  • Colors are layout from top to bottom. This is exactly how single-column list works.
  • When multi-column lists, colors are layout column after column and for each column is layout from top to bottom.

For the examples of list use case below, we have multi-column lists with three (3) columns.


Grids

Colors are layout row after row and for each row from left to right in normal mode (right to left mode not yet supported).
